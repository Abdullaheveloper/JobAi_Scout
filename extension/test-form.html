How to Test

  1. Load the extension in Chrome: chrome://extensions/ → Developer Mode → Load Unpacked → select the extension/ folder
  2. Open this page in Chrome
  3. Click the Job Form Fill extension icon → Sign in
  4. Click "Auto Fill This Form"
  5. Verify all fields are filled with your profile data
  6. Open DevTools Console (F12) to see [JobAI] logs
